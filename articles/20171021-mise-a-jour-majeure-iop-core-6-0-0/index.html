Mise à jour majeure de IoP Core en version 6.0.0

21 octobre 2017

Le client IoP Blockchain en version 6.0.0 est maintenant disponible !

Il s'agit d'une nouvelle version majeure, comprenant de nouvelles fonctionnalités, diverses corrections de bugs, une amélioration des performances, ainsi que la mise à jour des traductions.

Dû au changement du répertoire de données, un réindex complet de la blockchain est nécessaire. Il est également recommandé de supprimer tous les paquets des versions précédentes afin d'éviter toute confusion, bien que ça ne soit pas strictement nécessaire. Veuillez noter que vous ne pouvez pas utiliser les versions 5.0.1 et 6.0.0 de façon interchangeable avec le même répertoire de données.

Vous trouverez ci-joint les exécutables du nouveau portfeuille IoP :

  • IoP Core Wallet 6.0.0 pour Windows x64
  • IoP Core Wallet 6.0.0 pour Mac OS X
  • IoP Core Wallet 6.0.0 pour Ubuntu 16.04 amd64

Pour accéder aux autres versions disponibles et aux notes de mise à jour, consultez la page Github officielle du projet.
