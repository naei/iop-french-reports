Arrivée du wallet IoP en version 6.1.0

Nouveaux thèmes et mise à jour de la blockchain depuis Bitcoin Core 0.15

4 décembre 2017

Le portefeuille IoP (Core Wallet) vient d'être mis à jour en version 6.1.0 !

Les clients sont téléchargeables dès aujourd'hui :

  • IoP Core Wallet 6.1.0 pour Windows x64
  • IoP Core Wallet 6.1.0 pour Mac OS X
  • IoP Core Wallet 6.1.0 pour Ubuntu 16.04 amd64
Cette version introduit deux nouveaux thèmes appelés light theme et dark theme. Au démarrage, vous serez accueilli par notre nouvel écran de démarrage avant de retrouver l'interface de portefeuille standard que vous connaissez. Pour activer l'un des nouveaux thèmes, accédez à Preferences-Display et sélectionnez le dans le menu déroulant. Après un redémarrage du client, le portefeuille vous accueillera avec une nouvelle interface épurée. Le thème par défaut restera la norme jusqu'à ce que nous soyons complètement sûrs que tous les bugs de l'interface soient éliminés.

Toutes les mises à jour les plus récentes de Bitcoin Core 0.15 ont été implémentés.

Pour accéder aux autres versions disponibles et aux notes de mise à jour, consultez la page Github officielle du projet.
